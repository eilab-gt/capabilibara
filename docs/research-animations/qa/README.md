# Animation QA Notes

Use this directory for per-animation QA handoff files, named `<slug>_qa.md`.

Each QA note should record:

- the exact claim and values shown;
- the runtime and dev-page paths checked;
- whether the final frame is deterministic and poster-ready;
- reduced-motion behavior;
- review-mode anonymity status;
- export outputs, if any;
- remaining caveats or missing source artifacts.
